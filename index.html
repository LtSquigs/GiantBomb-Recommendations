1/11/2012 Update!

Hey guys, I have updated the code a bit to now take into account user ratings. I.E Ratings without a written review. Here is how it works.

Ratings are now considered for your own preferences when determining what kind of games you like. So if you rate things but don't write a review that will be considered when determining recommendations for yourself.

However, when determining what recommendations to give other people, only written reviews are considered. This is a technical limitation (there is no way through the api to get all the ratings for a game, only written reviews).

So basically, if you want your review to matter for recommending games to others, you need to write an actual review. If you just want to mark something as you liked it for your own recommendations then you can just use user ratings.
